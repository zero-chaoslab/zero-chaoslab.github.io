记录一些随笔、感想、思考

这里想留下一些还没有被整理成答案的东西。它们可能只是一天结束后的感想，读到一句话时停下来的念头，或者工作与生活之间忽然冒出来的小问题。

我希望这个地方不只记录结果，也记录过程。包括犹豫、改变主意、重新理解自己的时刻。很多真正重要的东西，往往不是在结论里发生，而是在反复思考的途中慢慢显现。

Notes, Impressions, and Thoughts

Here I want to leave behind a few things that have not yet been arranged into answers. Some may be end-of-day reflections, a thought that appeared while reading a sentence, or a small question that surfaced somewhere between work and life.

I want this space to record not only conclusions, but also process: hesitation, changed minds, and the moments when understanding quietly shifts. Many important things do not happen inside the final answer; they become visible along the way.

从零构建 Agent Memory Building Agent Memory from Scratch

Entry 002 Memory Workflow

从0开始自制一个0依赖、本地化的 Agent memory，支持跨 Agent 复用、自动提炼与召回、skill 路由和 memory 可视化。 Build a zero-dependency, localized Agent memory from scratch, with cross-Agent reuse, automatic refinement and recall, skill routing, and memory visualization.

本文会讲述如何从0开始自制一个0依赖、本地化的 Agent memory，让同一套 context / memory 可以在不同的 coding Agent 之间复用，并在用户与 coding Agent 交互的过程中实时、自动地被提炼、召回、修正。

它还会继续讨论如何通过 memory 路由 skill、如何把稳定的 workflow 自动转化为新的 skill，以及如何用 memory graph 和 recall trace 让整个 memory 结构变得可理解、可审计、可维护。

阅读全文

This note explains how to build a zero-dependency, localized Agent memory from scratch, so the same context / memory system can be reused across coding Agents and refined, recalled, and corrected automatically during user-Agent interaction.

It also follows how memory routes skills, how stable workflows can become new skills, and how memory graph plus recall trace visualization make the memory structure understandable, auditable, and maintainable.
